\CWHeader{Лабораторная работа \textnumero 4 \enquote{Разработка булевого индекса и булевого поиска по индекесу}}
В рамках данной работы была реализована подсистема индексирования и поиска по текстовой коллекции документов.
В отличие от предыдущих заданий, формальная формулировка задачи не была задана заранее, поэтому цель работы
определялась исходя из общей архитектуры информационно-поисковых систем и уже реализованных этапов
обкачки, очистки и токенизации документов.

Целью данной работы является разработка и реализация булевого инвертированного индекса, обеспечивающего:

\begin{itemize}
    \item добавление документов в индекс на основе их токенизированного представления;
    \item хранение для каждого термина списка документов, в которых он встречается;
    \item вычисление частоты термина по всей коллекции;
    \item обработку булевых поисковых запросов с операторами \texttt{AND}, \texttt{OR}, \texttt{NOT};
    \item сохранение и загрузку индекса;
\end{itemize}

Реализуемый индекс предназначен для работы с корпусом новостных и аналитических статей,
предварительно собранных поисковым роботом и очищенных от HTML-разметки.
